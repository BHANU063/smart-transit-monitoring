# Problem Statement

Urban public transport systems, specifically in Bengaluru, are highly utilized but critically under-optimized. Passengers face three core issues daily:
1. **Unpredictable Bus Timings**: Heavy city traffic makes standard timetables useless.
2. **Severe Overcrowding**: Passengers often wait for extended periods only to find that an arriving bus is too full to board.
3. **Lack of Centralized Information**: There is no reliable platform offering smart decision-making support (e.g., whether to wait for the next bus or take the current crowded one).

Concurrently, transit authorities face challenges because they lack unified, real-time data regarding specific route delays, passenger loads, and systematic fleet density. Without this data, optimizing resources and route planning is impossible. 

The goal of this project is to create an integrated system that captures location, tracks crowd sizes, calculates delay factors due to traffic, and provides actionable recommendations to passengers and valuable analytics to authorities.
